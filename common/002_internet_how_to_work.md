## 2. 브라우저 렌더링 과정에 대해 설명해주세요.

- HTML 마크업을 처리하고 DOM 트리를 빌드합니다.

- CSS 마크업을 처리하고 CSSOM 트리를 빌드합니다.

- DOM 및 CSSOM을 결합하여 렌더링 트리를 형성합니다.

- 렌더링 트리에서 레이아웃을 실행하여 각 노드의 기하학적 형태를 계산합니다.

- 개별 노드를 화면에 페인트합니다.
