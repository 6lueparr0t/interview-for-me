## Jenkins 사용 경험

- 젠킨스 파이프라인 사용

- view 탭으로 개발, QA, 운영으로 나눔

- 공통 함수는 groovy 파일로 만들어서 파이프라인 안에서 사용

- stage 별로 build, ssh connection, s3 upload/download, blue/green deploy 등의 작업을 했음

- 온프레미스 환경에서 이후 AWS 로 변경

- 젠킨스는 마스터 - 슬레이브 구조로 되어있고, AWS 를 위한 배포서버가 따로 있었음
